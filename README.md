# demo-task-manager
Demo Task Manager with a simple login, add task, add subtask, comment, close.

## Stack
Node JS 10
HAPI Framework
Angular 7.10
PostgreSql
Twitter Bootstrap
Google Fonts

The Requirement list:

User can login with google
User can add tasks
User can complete tasks
User leave comments for a task
A task can have sub tasks
